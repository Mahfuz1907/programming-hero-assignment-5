Question-1: What are the different ways to select an element in the DOM?

There are several ways to select elements in the DOM. getElementById selects an element by its unique id. getElementsByClassName retrieves all elements with a given class name. getElementsByTagName collects elements with a specified tag name. Then querySelector selects the first element that mathes a CSS selector and querySelectorAll gathers all matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML innerText and textContent are properties used to set the content of HTML elements, but they function differently. innerHTML allows us to access or modify the HTML content of an element. innerText sets only the visible text within an element. Meanwhile, textContent returns or sets the text content of an element.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to manage events for multiple child elements. When an event like a click occurs on an element, the event bubbles up to the parent element, where it can be captured and processed. This technique is efficient because it minimizes the number of event listeners and allows dynamic handling of elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in the DOM where an event starts at the deepest target element and then bubbles up through its parent elements. For example, if we click on a child element within a nested structure, the click event triggers on the child first and then sequentially on its parent, grandparent, and so on, until it reaches the root element.

Question-5: How do you create, add, and remove elements using JavaScript?

Creating, adding, and removing elements in the DOM with JavaScript involves several methods. We can create a new element using document.createElement, specifying the type of element needed. Once created, this element can be added to the DOM using methods like appendChild to add it as the last child of a parent element or insertBefore for more specific positioning. Removing elements is straightforward with the remove method for direct removal or removeChild when we need to remove a specific child from its parent element.
